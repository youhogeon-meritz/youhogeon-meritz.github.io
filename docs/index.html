처음부터 main(), 풍부한 구문, 동적 타이핑, 가비지 컬렉션, 어휘 범위, 일류 함수, 폐쇄, 클래스, 상속을 특징으로 하는 언어를 만듭니다. 모두 수천 줄의 깨끗하고 빠른 코드로 포장되어 있으며, 각 코드를 직접 작성하기 때문에 충분히 이해할 수 있습니다.
